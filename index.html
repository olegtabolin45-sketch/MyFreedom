Поддерживается формат Т-Банка (.xlsx). Прибыль и доходность (XIRR) считаются по сделкам из загруженного отчёта — для точности загрузите отчёты за весь период владения. Цены обновляются раз в 30 минут во время работы биржи; иностранные котировки и подключение брокеров по API — на следующих этапах.
